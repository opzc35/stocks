# 快速开始指南

## 🚀 启动应用

1. **安装依赖**
   ```bash
   npm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **访问应用**
   打开浏览器访问：http://localhost:5173/

## 📊 使用价格预警功能

### 设置你的第一个预警

1. **查看右侧预警面板**（🔔 价格预警）

2. **选择预警类型**
   - 突破 ▲：价格上涨到目标时提醒
   - 跌破 ▼：价格下跌到目标时提醒

3. **输入目标价格**
   - 例如：65000

4. **点击"添加"按钮**
   - 预警立即生效

5. **等待触发**
   - 当价格达到目标时，右上角会弹出通知
   - 预警面板中的预警状态会更新

### 测试预警功能

为了快速测试预警是否工作，你可以：

1. 查看当前 BTC 价格（例如：67,532.45）
2. 设置一个略低于当前价的"跌破"预警（例如：67,530）
3. 或设置一个略高于当前价的"突破"预警（例如：67,540）
4. 等待价格波动触发预警

## 📰 使用新闻展示功能

### 查看新闻

1. **新闻面板位置**：右侧下方（📰 新闻动态）

2. **筛选新闻**
   - 点击顶部按钮：全部 / 📈 利好 / 📉 利空 / 📰 中性

3. **查看新闻详情**
   - 点击任意新闻卡片
   - 左侧图表上对应的标记点会高亮

### 图表上的新闻标记

1. **查看标记点**
   - 图表上的彩色圆点表示新闻事件
   - 🟢 绿色 = 利好消息
   - 🔴 红色 = 利空消息
   - ⚪ 灰色 = 中性消息

2. **点击标记点**
   - 右侧新闻面板会滚动到对应新闻
   - 新闻卡片会高亮显示

## 📈 查看价格走势

### 开启机器人通知

1. **打开机器人设置**
   点击顶部导航栏的"🤖 机器人"按钮

2. **选择平台**
   - 📱 Telegram（推荐，最简单）
   - 💬 Discord
   - 💼 Slack
   - 💚 企业微信

3. **配置机器人**
   根据选择的平台填写配置信息

### Telegram 快速配置（最简单）

**步骤 1：创建机器人**
1. 在 Telegram 搜索 `@BotFather`
2. 发送 `/newbot`
3. 按提示设置名称
4. 复制 Bot Token

**步骤 2：获取 Chat ID**
1. 搜索你创建的机器人
2. 发送 `/start`
3. 访问：`https://api.telegram.org/bot你的Token/getUpdates`
4. 找到 `chat.id` 的值

**步骤 3：添加到应用**
1. 点击"+ 添加机器人"
2. 选择 Telegram
3. 填写名称、Bot Token、Chat ID
4. 点击"测试"确认
5. 启用机器人

### 测试机器人通知

1. 配置好机器人后
2. 设置一个容易触发的预警
3. 等待预警触发
4. 在社交平台查收通知

**示例：**
```
🚀 价格预警触发！

交易对: BTC/USDT
类型: 突破
目标价格: $70000.00
当前价格: $70125.50
时间: 2026/6/16 14:30:25
```

### 技术指标

1. **实时价格**：左侧第一个卡片显示当前价格和交易量

2. **价格图表**：显示历史价格走势和移动平均线
   - 绿色线：价格
   - 蓝色线：SMA(20)
   - 橙色线：SMA(50)

3. **技术指标**：显示 RSI、MACD、SMA 等指标

### 切换交易对

点击右上角的下拉菜单，选择不同的交易对：
- BTC/USDT
- ETH/USDT
- BNB/USDT

## 🔧 操作功能

### 同步历史数据

点击"📥 同步数据"按钮：
- 从交易所获取最近 7 天的历史数据
- 更新图表和技术指标

### 运行策略回测

点击策略回测按钮：
- 🎯 MA策略回测：使用移动平均线策略
- 📊 RSI策略回测：使用相对强弱指标策略

查看回测结果：
- 总收益
- 夏普比率
- 最大回撤
- 胜率
- 总交易次数

## 💡 使用技巧

### 预警设置技巧

1. **关键价位预警**
   - 在图表上找到支撑位和阻力位
   - 在这些位置设置预警

2. **分层预警**
   - 设置多个价格预警
   - 例如：65000、68000、70000
   - 形成完整的价格监控网

3. **及时管理**
   - 定期清理已触发的预警
   - 或点击 🔄 重置继续使用

### 新闻分析技巧

1. **关联分析**
   - 查看价格波动时附近的新闻
   - 分析新闻对价格的影响

2. **情绪追踪**
   - 筛选"利好"新闻，关注上涨机会
   - 筛选"利空"新闻，注意下跌风险

3. **时间对比**
   - 观察新闻发布时间与价格变化
   - 评估市场反应速度

## 🎯 实战场景

### 场景 1：突破交易

1. 观察图表，发现价格在 68000 附近震荡
2. 设置突破预警：68500
3. 查看新闻，确认市场情绪
4. 收到突破通知后，确认买入信号
5. 设置止盈预警：70000

### 场景 2：支撑位买入

1. 价格从高位回落
2. 在图表上找到支撑位：65000
3. 设置跌破预警：65000
4. 筛选利好新闻，评估反弹可能
5. 触发后在支撑位附近买入

### 场景 3：新闻驱动交易

1. 看到重大利好新闻（如机构增持）
2. 观察图表确认价格反应
3. 设置突破预警在下一个阻力位
4. 预警触发后跟进趋势
5. 设置止损预警保护利润

## 🔐 注意事项

### 预警功能

- ⚠️ 预警仅在应用打开时生效
- 🔄 刷新页面后预警会重置（未来将支持持久化）
- 🔊 音频通知需要浏览器权限

### 数据获取

- 📡 需要 Python 后端运行（默认 http://localhost:8000）
- 🌐 需要网络连接获取实时数据
- 💾 历史数据需要手动同步

### 浏览器兼容

- ✅ Chrome/Edge（推荐）
- ✅ Firefox
- ✅ Safari
- ⚠️ 部分功能可能在旧版浏览器中不可用

## 🐛 遇到问题？

### Python 后端未运行

如果看到"加载中..."一直不消失：

1. 确认 Python 后端已启动：
   ```bash
   cd python-engine
   python main.py
   ```

2. 检查端口是否正确（默认 8000）

### 预警不触发

1. 确认价格数据正常更新（查看实时价格）
2. 检查预警条件设置是否合理
3. 查看浏览器控制台是否有错误

### 新闻不显示

- 目前使用模拟数据
- 确保应用正常加载
- 未来可接入真实新闻 API
