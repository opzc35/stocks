# AI 交易助手快速使用指南

## 🚀 开始使用

### 1. 打开 AI 助手

点击顶部导航栏的 **"🧠 AI助手"** 按钮，AI 助手对话窗口会弹出。

### 2. 与 AI 对话

在底部输入框输入你的问题或需求，按回车或点击发送按钮。

## ✨ 功能特点

### 1. 智能理解

AI 能理解上下文和你的意图：

```
你: 创建一个策略
AI: 策略已创建...

你: 运行回测
AI: (理解是对刚创建的策略回测)
```

### 2. 自动执行

AI 不仅回答，还会自动执行操作：

```
你: 设置预警
  ↓
AI 自动配置
  ↓
预警立即生效
```

### 3. 实时反馈

通过动作卡片看到执行状态：

- ⏳ 待执行
- 🔄 执行中
- ✅ 已完成
- ❌ 失败

### 4. 持续对话

AI 会记住本次对话的内容：

```
你: 创建策略
AI: 策略已创建...

你: 优化它
AI: (知道是优化刚创建的策略)
```

## 💡 使用技巧

### 1. 清楚表达

**好的：**
- "帮我创建一个基于 RSI 的策略"
- "运行最近 30 天的回测"
- "价格突破 70000 时提醒我"

**不好的：**
- "做点什么"
- "看看"
- "那个"

### 2. 提供细节

当需要更多信息时，提供具体数字：

```
AI: 您想设置什么价格？
你: 70000 美元
```

### 3. 分步操作

对于复杂任务，可以分步进行：

```
你: 创建策略
AI: 完成

你: 运行回测
AI: 完成

你: 优化参数
AI: 完成
```

### 4. 确认理解

不确定时，让 AI 确认：

```
你: 你是说短期用 18 天均线吗？
AI: 是的，18 天短期均线
```

## 🔧 高级用法


### 智能建议

询问建议：

```

```

## ⚠️ 注意事项

1. **AI 建议仅供参考**
   - 不构成投资建议
   - 请自行判断决策

2. **AI 不会直接交易**
   - 只创建策略和预警
   - 不会执行真实买卖

3. **网络连接**
   - 需要网络才能使用
   - 响应速度取决于网络

4. **对话隐私**
   - 对话仅保存在本地
   - 刷新页面会清空

## 🆘 常见问题

### Q: AI 能帮我自动交易吗？
**A:** 不能。AI 只能创建策略、设置预警，不会自动执行真实交易。

### Q: AI 的建议准确吗？
**A:** AI 基于技术分析给出参考建议，但市场有风险，请谨慎决策。

### Q: 可以用语音对话吗？
**A:** 目前仅支持文字输入，未来将支持语音。

### Q: 对话会保存吗？
**A:** 当前会话会保存，刷新页面后清空。未来将支持历史记录。

### Q: 支持哪些语言？
**A:** 目前主要支持中文，也可以理解简单英文。
