# 🖤 Vercel 风格设计升级

## 设计理念

从奢华的毛玻璃设计转变为 **Vercel 式极简科技风格**：
- **Less is More** - 极简主义
- **黑白为主** - 经典配色
- **边框定义** - 清晰分隔
- **细节精致** - 微妙交互

---

## 🎨 设计系统

### 色彩系统

#### 主色调
```css
--background: #000000  /* 纯黑背景 */
--foreground: #ffffff  /* 纯白文字 */
--border: #27272a      /* 深灰边框 */
```

#### 灰度阶梯
- 50 层灰度，从纯黑到纯白
- 提供细腻的层次感
- 无彩色，专注内容

#### 强调色
```css
--accent: #0070f3      /* Vercel 蓝 */
```
- 仅用于重要操作
- 链接、按钮、焦点状态

### 视觉特点

#### ❌ 移除的元素
- ❌ 渐变色
- ❌ 毛玻璃效果
- ❌ 发光阴影
- ❌ 粒子背景
- ❌ 复杂动画

#### ✅ 保留的元素
- ✅ 简洁边框
- ✅ 微妙阴影
- ✅ 平滑过渡
- ✅ 清晰层次
- ✅ 响应式布局

## 📐 布局规范

### 间距系统
```
--spacing-xs:  0.25rem  (4px)
--spacing-sm:  0.5rem   (8px)
--spacing-md:  1rem     (16px)
--spacing-lg:  1.5rem   (24px)
--spacing-xl:  2rem     (32px)
--spacing-2xl: 3rem     (48px)
```

### 圆角系统
```
--radius-sm:   0.25rem  (小按钮)
--radius-md:   0.5rem   (标准)
--radius-lg:   0.75rem  (卡片)
--radius-full: 9999px   (圆形)
```

### 阴影系统
```
--shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.3)
--shadow-md: 0 4px 8px rgba(0, 0, 0, 0.4)
--shadow-lg: 0 8px 16px rgba(0, 0, 0, 0.5)
```
- 仅在悬停时显示
- 轻微提升感

## 🧩 组件风格

### Header（导航栏）
```
背景: 半透明黑色 + 12px 模糊
边框: 底部 1px 细线
高度: 64px
定位: sticky
```

### 标签按钮
```
默认: 透明背景 + 灰色文字
悬停: 深灰背景
激活: 深灰背景 + 边框
```

### 卡片
```
背景: 纯黑
边框: 1px 深灰
圆角: 0.75rem
悬停: 边框变亮 + 轻微阴影
```

### 按钮
```
Primary:   白色背景 + 黑色文字
Secondary: 透明背景 + 边框
Success:   蓝色背景
Ghost:     无边框，仅文字
```

### 输入框
```
背景: 纯黑
边框: 1px 深灰
焦点: 蓝色边框 + 轻微发光
```

## 💫 交互设计

### 过渡时间
```
--transition-fast: 150ms  (默认)
--transition-base: 200ms  (按钮)
--transition-slow: 300ms  (动画)
```

### 缓动函数
```css
cubic-bezier(0.4, 0, 0.2, 1)
```
- 标准 ease-in-out
- 平滑自然

### 悬停效果
- **边框**: 变亮
- **背景**: 深灰色
- **提升**: -2px 上浮
- **阴影**: 轻微显示

### 焦点效果
- **边框**: 蓝色
- **发光**: 3px 蓝色光圈
- **清晰**: 易于识别

```css

```

### 适配策略
- 导航栏高度调整
- 按钮尺寸缩小
- 间距压缩
- 卡片单列

## 🖥️ Vercel 式特征

### 1. 极简主义
- 黑白为主色
- 边框分隔内容
- 大量留白
- 无多余装饰

### 2. 清晰层次
```
背景: 纯黑 (#000000)
卡片: 有边框
悬停: 边框变亮
```

### 3. 微妙交互
- 150ms 快速响应
- 轻微上浮 (-2px)
- 边框状态变化
- 无夸张效果

### 4. 等宽数字
```css
font-variant-numeric: tabular-nums;
```
- 数字对齐
- 易于阅读
- 专业感

### 5. 系统字体
```css
-apple-system, BlinkMacSystemFont, 'Segoe UI', ...
```
- 原生体验
- 快速加载
- 一致性高

## 📝 使用示例

### 创建卡片
```tsx
<div className="card">
  <div className="card-header">
    <h2 className="card-title">标题</h2>
  </div>
  内容...
</div>
```

### 创建按钮
```tsx
<button className="btn btn-primary">主要操作</button>
<button className="btn btn-secondary">次要操作</button>
<button className="btn btn-ghost">幽灵按钮</button>
```

### 创建统计卡
```tsx
<div className="stat-card">
  <div className="stat-label">标签</div>
  <div className="stat-value">$1,234.56</div>
  <div className="stat-change positive">↗ +5.2%</div>
</div>
```

## 🎨 自定义主题

### 修改强调色
```css
:root {
  --accent: #0070f3;  /* 改为你的品牌色 */
}
```

### 修改边框颜色
```css
:root {
  --border: #27272a;      /* 默认 */
  --border-hover: #3f3f46; /* 悬停 */
}
```

### 调整圆角
```css
:root {
  --radius-lg: 0.75rem; /* 卡片圆角 */
}
```
